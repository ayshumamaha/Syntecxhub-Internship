# File Organizer Script

## Project Overview

This project is a Python automation tool that scans a folder and automatically organizes files into subfolders based on their file extensions. It also supports dry-run mode, logging, and filename collision handling for safe and efficient file management.

## Features

* Scan folders automatically
* Organize files by extension
* Create subfolders automatically
* Handle filename collisions
* Dry-run mode
* Activity logging
* Uses `os` and `shutil`
* Simple and modular design

## Technologies Used

* Python
* os
* shutil
* logging

## Files

* `file_organizer.py` – Main Python program
* `organizer.log` – Stores file movement logs

## Run

```bash
python file_organizer.py
```
