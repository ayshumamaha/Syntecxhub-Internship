# To-Do List Manager

## Project Overview

This project is a menu-driven command-line application that helps users manage daily tasks efficiently. It allows users to add, view, search, delete, and mark tasks as completed while storing all task information in a JSON file for persistent storage.

## Features

* Add new tasks
* View all tasks
* Mark tasks as completed
* Delete tasks
* Search tasks
* Display task statistics
* Persistent JSON storage
* Input validation and exception handling

## Technologies Used

* Python
* json (built-in library)
* os
* datetime

## Files

* `todo_list_manager.py` – Main Python program
* `tasks.json` – Stores task records

## Run

```bash
python todo_list_manager.py
```
